# 🎯 Digital Tools Buying Website

This is a modern and responsive digital tools marketplace where users can explore different tools and add them to their cart easily.

---

## 📖 Project Overview

This project was built as part of an assignment to practice React fundamentals, state management, and UI design implementation from Figma.

The goal was to create a clean, user-friendly interface where users can browse digital products and manage a shopping cart efficiently.

---

## 🛠️ Technologies Used

- React.js
- Tailwind CSS
- JavaScript (ES6+)
- React Toastify
- JSON (Mock Data)

---

## ✨ Features

- 🛒 Add to cart with live count update in navbar
- 🔄 Toggle between Products and Cart section
- ❌ Remove individual items from cart
- 💳 Checkout button clears the cart instantly
- 🔔 Toast notifications for all actions

---

## 📱 Responsiveness

The website is fully responsive and works smoothly across mobile, tablet, and desktop devices.
